    (2) Write the following symbolic statement in words:
    
    (i) 4/3 ∈ Q
    Solution:
    4/3 belong to the set Q.
    or
    4/3 is an element of the set Q.
   
   (ii) –2 ∉ N
   Solution:
    –2 does not belong to the set N or –2 is not an element of the set N.
   
    (iii) P = { p/p is an odd number }
    Solution:
     P is the set of all p such that p is an odd number.
